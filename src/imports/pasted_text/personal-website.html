Build a minimal one-page personal website. GENERAL IDEA The entire website should look like a minimal computer monitor displaying a web browser. Inside the monitor is a simple browser window with tabs, similar to Google Chrome. The page should NOT scroll. Instead, users switch content using browser tabs. The layout should be centered on the screen and responsive. STYLE Minimal, clean, modern. No heavy animations, no unnecessary UI elements. Inspired by a nature-themed color palette: dark forest green, moss green, muted teal, and soft earthy browns. Color palette example: background: #0f1a16 monitor frame: #1b2a24 browser background: #16221d accent green: #6f9f84 light text: #e6f0ea muted text: #a6b8ad highlight: #7fbf9a Fonts: Use a clean sans-serif font. Prefer: Inter or JetBrains Mono for small UI elements. GENERAL LAYOUT Page structure: centered monitor frame inside the monitor: browser window inside browser window: tab bar + content area Example visual structure: [ screen background ] monitor frame ┌──────────────────────┐ │ │ │ browser window │ │ ┌────────────────┐ │ │ │ tabs │ │ │ │ │ │ │ │ page content │ │ │ │ │ │ │ └────────────────┘ │ │ │ └──────────────────────┘ MONITOR DESIGN Very minimal. Thin rounded border. Subtle shadow. Do NOT make the monitor too bulky. BROWSER WINDOW Include: • tab bar • fake address bar • minimal browser controls Example tabs: Welcome About Projects Philosophy Connect Tabs should switch content using simple JavaScript (no page reload). TAB 1 — WELCOME Content: Title: Hi, I'm Quillpy Subtitle: Curious mind exploring code, systems, and ideas. Small description: I enjoy programming, Linux systems, and thinking about how things work. Currently learning and building small projects while exploring engineering and creative technology. Optional: typing cursor animation after the title. TAB 2 — ABOUT Short and minimal. Content example: Alias: Quillpy I like building things with code and experimenting with systems. Interests: • programming • Linux customization • chess • game mechanics • creative engineering ideas I enjoy simple tools, minimal design, and learning by building. TAB 3 — PROJECTS Show 3–4 minimal project cards. Example projects: FastCR A lightweight fast code runner. Linux Setup Custom Arch Linux KDE environment with minimal design. Dotfiles Minimal configs for my Linux setup. Experimental Ideas Small coding experiments and tools. Cards should be simple rectangles with hover effect. Click opens GitHub link. TAB 4 — PHILOSOPHY Short ideas / principles. Examples: Simplicity beats complexity. Build things that make you curious. Understand systems instead of memorizing tools. Technology should feel elegant. Keep this section very minimal. TAB 5 — CONNECT Minimal links with icons. GitHub Chess.com Email Layout should be centered and clean. INTERACTION Only subtle interactions: • tab switch animation • hover highlight • optional blinking cursor in welcome section No heavy animations. TECH Use simple stack: HTML CSS Vanilla JavaScript Avoid frameworks unless necessary. Performance should be fast. EXTRA DETAILS Fake browser address bar should change visually when tabs switch: example: quillpy.dev/welcome quillpy.dev/about quillpy.dev/projects But this is only visual, not real routing. FINAL GOAL A minimal, elegant personal website that feels like opening a browser inside a monitor. It should be simple, clean, interactive, and memorable. Avoid clutter. Avoid large UI elements. Keep everything light and minimal.
